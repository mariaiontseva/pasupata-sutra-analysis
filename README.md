# Pāśupata Sūtra - Comprehensive Scholarly Analysis

Interactive scholarly translation table for the Pāśupata Sūtra commentary with auto-save functionality.

## Features

- ✏️ **Editable Cells**: Both "Word-by-Word Analysis" (blue) and "Literal Translation" (yellow) columns are editable
- 💾 **Auto-save**: Changes are automatically saved every 60 seconds
- 🔄 **Browser Backup**: Changes are backed up to browser localStorage immediately
- 📝 **5-Column Analysis**:
  - Sanskrit Text
  - Word-by-Word Analysis (editable)
  - Literal Translation (editable)
  - Interpretative Meaning
  - Literary Translation

## How to Use

1. Open the page: https://[your-username].github.io/pasupata-sutra-analysis/
2. Click on any blue (Word-by-Word) or yellow (Literal Translation) cell to edit
3. Your changes will be saved automatically every 60 seconds
4. A backup is saved to your browser immediately after each edit

## Auto-save Setup

When you first open the page:
1. A dialog will appear asking you to select a file for auto-save
2. Choose where to save the HTML file on your computer
3. After selecting, you'll see: "✓ Auto-save enabled! Changes will be saved every 60 seconds."

## Browser Requirements

- Works best in Chrome, Edge, or Opera (latest versions)
- Requires File System Access API support for auto-save to file
- Falls back to localStorage-only mode in other browsers

## Academic Use

This tool is designed for dissertation research on the Pāśupata Sūtra, providing an interactive platform for refining translations and grammatical analysis.
